# Unspark — design brief package

This is a brief for **claude.ai/design** to generate a native macOS app interface for Unspark — a local AI sparkle remover for Google Gemini / Nano Banana images.

## Entry point

Read **BRIEF.md** first. It's the full brief, structured in 10 sections.

## Folders

- `brand/` — current icon, voice examples
- `current_state/` — what the app looks like today (the bad UX we're replacing)
- `reference/` — visual DNA references
- `samples/` — actual before/after pair (Gemini map → LaMa-cleaned)

## How to use

1. In **claude.ai/design**: Connect GitHub → install `claude-design-import` on repo `L-tune/unspark`
2. Branch: `claude-design-brief`
3. Paste the starter prompt from §9 of BRIEF.md
