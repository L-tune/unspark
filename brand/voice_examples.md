# Unspark — voice examples (extended)

## Tagline candidates

- Strip the sparkle.
- Local sparkle remover for Gemini.
- No cloud. No signup. No nonsense.
- Drop. Done.

## Window copy (по states)

### Idle
- Hero: "Drop a Gemini image."
- Subtext: "We strip the sparkle. Locally."
- Bottom status: "Ready · Local · Offline"

### Drag-over
- Hero (replaces idle): "Drop now."
- Subtext stays: "We strip the sparkle. Locally."

### Processing
Sequence (rotates every 2.5s):
1. "Reading the sparkle…"
2. "Erasing…"
3. "Almost there…"

### Result
- Top stat: "Done in 7.2 seconds." (replace number from real timing)
- Caption: "Local LaMa · no upload"
- Primary CTA: "Save…"
- Secondary CTA: "Process more"
- Tertiary: "Reveal in Finder" (icon only)

### Batch
- Header: "Processing 12 images…"
- Per-row status: queued / processing / done / error
- Footer counter: "9 of 12 done"
- Final: "12 images, no sparkles."

### Error
- File unsupported: "This isn't an image we can read."
- Detection failed: "Couldn't find a sparkle to remove."
- Action: "Try anyway with a manual mask"

### About
One-liner: "A local sparkle remover for Gemini images. No cloud, no signup, no nonsense."

Credits block:
- "Built by Alexey Evdokimov"
- "evdokimov.ai" (link)

Tech block:
- "Powered by LaMa large-mask inpainting"
- "Free and open source"

## What we DON'T say

- "Welcome to Unspark!"
- "Powered by AI"
- "Intelligent watermark removal"
- "Cutting-edge / revolutionary / breakthrough"
- "Seamless / effortless / smart"
- "Get started" → just say "Drop"
- "We're excited to / honored to / delighted to"
- Anything ending with "and more"
- "Made with love"
- "Crafted with care"

## Reference voice

Linear release notes — direct, terse, factual.
Raycast launchers — smart, slightly warm, no exclamation marks.
Apple's early iOS — declarative ("Designed for iPhone").
